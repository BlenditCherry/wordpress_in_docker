## Dependencies
- [Docker](https://docs.docker.com/get-docker/)
- [Git Bash](https://git-scm.com/downloads)

## How to Build
``` bash
$ cp example.env .env
$ docker-compose build --no-cache
```

## How to Run
``` bash
$ docker-compose up -d
```
